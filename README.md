## 页面功能

1. 分数表的增加，删除，修改

2. 用户登录，注册（涉及session 添加，删除，分页）

3. 获取验证码

4. 上传图片到静态资源 www/static/upload

5. 七牛上传图片 

6. 用户登录后可修改头像

7. 文章添加（待续...）
